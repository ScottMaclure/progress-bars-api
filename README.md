# progress-bars-api

http://progress-bars-api.gotest.maclure.info/bars

* 2-5 bars, between 10-90
* 4 buttons, 2x positive, 2x negative, between +/- 5-50

Example

```
{

    "bars": [
        16,
        17,
        90
    ],
    "buttons": [
        12,
        50,
        -33,
        -47
    ]

}
```
